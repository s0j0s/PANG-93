# 팡(PANG) 게임 기획서

> 원작: 미첼(Mitchell Corporation), 1989년 아케이드 출시  
> 북미판 제목: Buster Bros. (캡콤 유통)

---

## 게임 개요

화면에 등장하는 모든 풍선(버블)을 작살로 터뜨려 스테이지를 클리어하는 액션 게임.  
전 세계 17개 실제 지역을 배경으로 총 50개 스테이지가 구성되어 있으며, 2인 동시 협력 플레이를 지원한다.

---

## 스테이지 구성

| 항목 | 내용 |
|------|------|
| 전체 지역 수 | 17개 (이스터섬, 파리, 후지산 등 세계 유명 명소) |
| 전체 스테이지 수 | 50개 |
| 지역당 스테이지 | 대부분 3스테이지 (마지막 지역만 2스테이지) |
| Mission 1 배경 | 일본 후지산 |

---

## 핵심 게임플레이

### 풍선(버블) 시스템

- 풍선은 **4단계 크기**로 존재 (대 → 중 → 소 → 극소)
- 작살에 맞으면 두 개의 더 작은 풍선으로 **분열** (최대 3회)
- 가장 작은 풍선은 맞으면 **소멸**
- 풍선은 바닥에서 튀어 오르며 좌우로 이동하는 **포물선 궤적**

### 클리어 / 게임 오버 조건

- **스테이지 클리어**: 화면의 모든 풍선 제거
- **라이프 감소**: 어떤 크기의 풍선에 닿아도 라이프 1 감소
- **게임 오버**: 모든 라이프 소진 (시작 라이프 3~5개, 딥스위치 설정에 따라 상이)

---

## 플레이어 조작

- **이동**: 좌우 수평 이동
- **공격**: 작살(Harpoon) 발사 — 화면 상단까지 수직으로 뻗어 고정됨

---

## 무기 시스템

| 무기 | 설명 |
|------|------|
| Wire (기본) | 작살 1발, 화면 상단에 고정되어 유지 |
| Double Wire | 작살 2발 동시 발사 |
| Power Wire | 갈고리 방식으로 즉시 수거 가능 |
| Vulcan Missile | 연속 발사 (고정 유지 없이 바로 사라짐) |

무기는 풍선 처치 시 드롭되는 **아이템**으로 획득.

---

## 파워업 아이템

| 아이템 | 효과 |
|--------|------|
| 모래시계 (Hourglass) | 풍선 이동 속도 감소 |
| 시계 (Clock) | 풍선 완전 동결 |
| 다이너마이트 (Dynamite) | 화면의 모든 풍선을 최소 크기로 축소 |
| 포스 필드 (Force Field) | 일정 시간 무적 |

---

## 멀티플레이

- **2인 동시 협력 플레이** 지원
- 두 플레이어가 같은 화면에서 독립적으로 작살을 발사

---

## 구현 범위 (이번 프로젝트)

이번 프로젝트에서는 원작의 전체 50스테이지 중 **Mission 1 (3스테이지)** 을 구현 목표로 한다.

- 후지산 배경 스테이지 3개
- 기본 풍선 분열 시스템
- 단일 플레이어 기준 구현
