# PLAN.md — Phase별 개발 목표

각 Phase는 **직접 브라우저에서 실행하고 확인할 수 있는 단위**로 구성된다.  
Phase 완료 시 고객이 직접 실행하여 체크포인트를 확인하고 피드백을 전달한다.

---

## Phase 1 — 메인 화면

**목표**: 게임을 켰을 때 가장 먼저 보이는 메인 화면을 만든다.  
게임 로직은 전혀 없고, 화면 구성과 버튼 UI만 완성하는 단계다.

### 구현 내용
- 게임 타이틀 **PANG** 표시
- 메뉴 버튼 3개: `GAME START` / `HOW TO PLAY` / `QUIT`
- 버튼에 마우스 오버 시 하이라이트 효과
- `QUIT` 버튼 클릭 시 "종료하시겠습니까?" 확인 메시지
- `HOW TO PLAY` 클릭 시 조작 방법 안내 팝업 (← → 이동, Space 발사)
- `GAME START` 클릭 시 "준비 중입니다" 또는 빈 화면으로 이동 (다음 Phase에서 채움)

### 고객 체크포인트

- [x] 버튼 3개가 보이고 마우스를 올리면 반응하는가?
- [x] `HOW TO PLAY` 를 클릭하면 조작 안내가 나오는가?
- [x] `QUIT` 버튼이 클릭 시 확인 동작을 하는가?
- [x] 전체적인 화면 레이아웃이 자연스러운가?

---

## Phase 2 — 게임 캔버스 & 플레이어 등장

**목표**: `GAME START`를 누르면 게임 화면으로 전환되고, 플레이어 캐릭터가 화면에 서 있다.  
아직 아무것도 움직이지 않아도 된다.

### 구현 내용
- 메인 화면 → 게임 화면 전환
- 고정 해상도 게임 캔버스 렌더링 (배경 포함)
- 플레이어 캐릭터가 화면 하단 중앙에 표시
- 후지산 배경 이미지 또는 단순 배경색 적용

### 고객 체크포인트
- [x] `GAME START` 클릭 후 게임 화면으로 넘어가는가?
- [x] 플레이어 캐릭터가 화면에 보이는가?
- [x] 배경과 캐릭터가 어색하지 않게 배치되어 있는가?

---

## Phase 3 — 플레이어 이동

**목표**: 키보드로 플레이어를 좌우로 움직일 수 있다.

### 구현 내용
- `←` `→` 키로 플레이어 좌우 이동
- 화면 경계에서 이동 제한 (벽 통과 불가)
- 이동 속도 적용

### 고객 체크포인트
- [x] 방향키로 캐릭터가 부드럽게 좌우 이동하는가?
- [x] 화면 끝에서 더 이상 나가지 않는가?
- [x] 이동 속도가 너무 빠르거나 느리지 않은가?

---

## Phase 4 — 풍선 등장 및 바운스

**목표**: 풍선이 화면에 등장하여 물리 법칙대로 튀어다닌다.  
아직 풍선을 터뜨릴 수 없고, 충돌 판정도 없다.

### 구현 내용
- 대형 풍선 1개가 화면에 등장
- 포물선 궤적으로 위아래 바운스
- 좌우 이동하며 벽에서 반사
- 풍선이 플레이어를 그냥 통과함 (충돌 없음)

### 고객 체크포인트
- [x] 풍선이 화면에 나타나는가?
- [x] 풍선이 바닥과 벽에서 자연스럽게 튀어오르는가?
- [x] 풍선의 바운스 움직임이 원작 팡과 유사하게 느껴지는가?

---

## Phase 5 — 작살 발사

**목표**: 스페이스 키를 누르면 작살이 발사된다.  
풍선을 맞혀도 아직 아무 일도 일어나지 않는다.

### 구현 내용
- `Space` 키로 작살 발사 (수직 상방)
- 작살이 천장에 닿을 때까지 뻗어 고정 유지 후 사라짐
- 작살이 화면에 있는 동안 재발사 불가
- 작살 시각적 표현 (선 또는 스프라이트)

### 고객 체크포인트
- [x] 스페이스를 누르면 작살이 위로 뻗는가?
- [x] 천장까지 닿은 후 자연스럽게 사라지는가?
- [x] 작살이 있는 동안 스페이스를 누르면 기존 작살이 교체 발사되는가?

---

## Phase 6 — 풍선 분열

**목표**: 작살이 풍선에 닿으면 풍선이 터지며 분열된다.  
팡 게임의 핵심 메커니즘이 완성되는 단계다.

### 구현 내용
- 작살과 풍선 충돌 감지
- 풍선 분열: 대 → 중+중 → 소+소 → 극소+극소 → 소멸
- 크기별 튀어오르는 높이와 이동 속도 차등 적용
- 분열 시 두 풍선이 좌우로 퍼지는 연출

### 고객 체크포인트
- [ ] 작살이 풍선에 닿으면 두 개로 나뉘는가?
- [ ] 4단계 분열이 모두 정상 동작하는가?
- [ ] 극소 풍선은 맞으면 사라지는가?
- [ ] 작은 풍선일수록 빠르고 낮게 튀는가?

---

## Phase 7 — 라이프 시스템

**목표**: 풍선에 닿으면 다치고, 라이프가 소진되면 게임 오버가 된다.

### 구현 내용
- 플레이어와 풍선 충돌 감지
- 충돌 시 라이프 1 감소 (시작 라이프 3개)
- 라이프 소진 시 게임 오버 화면 표시
- 게임 오버 화면에서 메인으로 돌아가기 버튼

### 고객 체크포인트
- [ ] 풍선에 닿으면 반응(깜빡임 등)이 있고 라이프가 줄어드는가?
- [ ] 라이프 3개를 모두 잃으면 게임 오버 화면이 뜨는가?
- [ ] 게임 오버에서 메인 화면으로 돌아갈 수 있는가?

---

## Phase 8 — 스테이지 클리어 및 Mission 1 구성

**목표**: 모든 풍선을 처치하면 다음 스테이지로 넘어가고, Mission 1 전체 3스테이지가 완성된다.

### 구현 내용
- 화면의 모든 풍선 소멸 시 스테이지 클리어
- 스테이지별 초기 풍선 배치 정의
  - Stage 1: 대형 1개
  - Stage 2: 대형 1개 + 중형 1개
  - Stage 3: 대형 2개
- Stage 3 클리어 시 Mission 1 완료 화면

### 고객 체크포인트
- [ ] 모든 풍선을 없애면 스테이지가 넘어가는가?
- [ ] Stage 1 → 2 → 3으로 갈수록 풍선 수가 늘어나는가?
- [ ] Stage 3 클리어 후 완료 화면이 나오는가?
- [ ] Stage 1 → 2 → 3 순서로 난이도가 체감되는가?

## Phase 9 — HUD (게임 내 정보 표시)

**목표**: 플레이 중 현재 상태를 화면에서 바로 확인할 수 있다.

### 구현 내용
- 현재 라이프 표시 (아이콘 또는 숫자)
- 현재 스테이지 번호 표시
- 점수 표시 (실시간 갱신)
- 점수 시스템: 극소(100pt) / 소(50pt) / 중(30pt) / 대(10pt)

### 고객 체크포인트
- [ ] 화면에서 라이프·스테이지·점수를 한눈에 확인할 수 있는가?
- [ ] 풍선을 터뜨릴 때마다 점수가 올라가는가?
- [ ] 작은 풍선을 터뜨릴수록 더 높은 점수가 오르는가?

## Phase 10 — 파워업 아이템

**목표**: 풍선을 처치하면 아이템이 드롭되고, 줍는 순간 효과가 발동된다.

### 구현 내용
- 풍선 처치 시 랜덤 아이템 드롭 (일정 확률)
- 아이템이 바닥으로 낙하
- 플레이어가 아이템에 닿으면 획득
- 모래시계(감속) / 시계(동결) / 다이너마이트(전체 극소화) / 포스 필드(무적) 효과 구현
- 아이템 효과 시각적 피드백 (색상 변화, 아이콘 표시)

### 고객 체크포인트
- [ ] 풍선을 터뜨리면 가끔 아이템이 떨어지는가?
- [ ] 아이템을 줍는 순간 효과가 발동되는가?
- [ ] 모래시계/시계 효과가 눈에 보이게 차이가 나는가?
- [ ] 다이너마이트를 먹으면 화면 풍선이 모두 극소로 변하는가?

---

## Phase 11 — 무기 시스템

**목표**: 다양한 무기를 획득하여 발사 방식을 바꿀 수 있다.

### 구현 내용
- 무기 아이템 드롭 (Double Wire / Power Wire / Vulcan Missile)
- 무기별 발사 동작 구현
  - Double Wire: 좌우 2방향 동시 발사
  - Power Wire: 즉시 수거 후 재발사 가능
  - Vulcan Missile: 연속 발사, 고정 없이 관통
- 현재 무기 HUD 표시

### 고객 체크포인트
- [ ] 무기 아이템을 주우면 발사 방식이 바뀌는가?
- [ ] Double Wire는 두 방향으로 동시에 발사되는가?
- [ ] Vulcan Missile은 빠르게 연속 발사되는가?
- [ ] 현재 어떤 무기를 들고 있는지 화면에서 확인되는가?

## 진행 순서 요약

```
Phase 1  ── 메인 화면 (타이틀 + 버튼 UI)
    ↓
Phase 2  ── 게임 캔버스 + 플레이어 등장
    ↓
Phase 3  ── 플레이어 이동
    ↓
Phase 4  ── 풍선 등장 + 바운스
    ↓
Phase 5  ── 작살 발사
    ↓
Phase 6  ── 풍선 분열
    ↓
Phase 7  ── 라이프 시스템 + 게임 오버
    ↓
Phase 8  ── 스테이지 클리어 + Mission 1 완성
    ↓
Phase 9  ── HUD + 점수
    ↓
Phase 10 ── 파워업 아이템
    ↓
Phase 11 ── 무기 시스템
```

각 Phase 완료 후 고객이 직접 브라우저에서 실행하여 체크포인트를 확인하고,  
피드백을 반영한 뒤 다음 Phase로 진행한다.
