# Mission 1 — 후지산 (일본)

## 개요

| 항목 | 내용 |
|------|------|
| 배경 | 일본 후지산 |
| 스테이지 수 | 3개 (Stage 1 ~ Stage 3) |
| 난이도 | 입문 — 게임의 기본 메커니즘을 익히는 구간 |

---

## 스테이지별 구성

### Stage 1
- **등장 풍선**: 대형(Large) 풍선 1~2개
- **목표**: 분열 메커니즘 체험
- **특징**: 풍선 수가 적고 이동 패턴이 단순함. 작살 조작과 풍선 분열을 익히는 튜토리얼 역할

### Stage 2
- **등장 풍선**: 대형 1개 + 중형(Medium) 1~2개
- **목표**: 분열된 풍선 동시 처리
- **특징**: 분열 후 여러 개의 풍선이 동시에 존재하는 상황 경험. 이동 경로 예측이 필요해짐

### Stage 3
- **등장 풍선**: 대형 2개 또는 중형 복수 개
- **목표**: 복수 풍선 동시 관리
- **특징**: Mission 1의 마무리 스테이지. 풍선 수가 늘어나며 움직임이 겹치기 시작. 이후 Mission으로 이어지는 난이도 브릿지 역할

---

## 난이도 설계 방향

```
Stage 1 ─── 단순 분열 체험
    ↓
Stage 2 ─── 분열 후 복수 풍선 처리
    ↓
Stage 3 ─── 복수 풍선 동시 관리 + 이동 예측
```

- 파워업 아이템은 Stage 1부터 드롭될 수 있으나 **의존하지 않아도 클리어 가능**한 수준으로 설계
- 무기 드롭은 Stage 2 이후부터 등장 (선택 사항)
- 시간 제한 없음 (플레이어가 룰을 익히는 구간이므로)

---

## 제약 조건

- 플레이어 이동: 좌우만 가능 (점프 없음)
- 작살: 동시에 1발만 존재 가능
- 라이프: 전역 공유 (스테이지 간 이월)

---

## 클리어 기준

각 스테이지에서 화면의 **모든 풍선 소멸** 시 다음 스테이지로 진행.  
Stage 3 클리어 시 Mission 1 완료 → (추후) Mission 2로 이동.
