# 게임 룰 상세

## 기본 목표

화면에 등장하는 **모든 풍선을 작살로 터뜨려** 스테이지를 클리어한다.

---

## 풍선(버블) 시스템

### 크기 단계
풍선은 총 4단계 크기로 존재하며, 맞을 때마다 두 개의 작은 풍선으로 분열된다.

```
[대]  →  [중] + [중]
[중]  →  [소] + [소]
[소]  →  [극소] + [극소]
[극소] →  소멸
```

### 이동 패턴
- 풍선은 **포물선 궤적**으로 바닥에서 튀어오른다
- 좌우 방향으로 이동하며 벽에 닿으면 반대 방향으로 반사
- 크기가 작을수록 **튀어오르는 높이가 낮고 이동 속도가 빠름**

---

## 플레이어 조작

| 키 | 동작 |
|----|------|
| `←` `→` | 좌우 이동 |
| `Space` | 작살 발사 |

- 작살은 **수직 위로** 발사되며 천장에 닿을 때까지 고정 유지
- 한 번에 **1발**만 발사 가능 (기본 무기 기준)
- 작살이 화면에 있는 동안 재발사 불가

---

## 무기 시스템

풍선 처치 시 무기 아이템이 드롭될 수 있으며, 획득 시 현재 무기를 교체한다.

| 무기 | 발사 방식 | 특징 |
|------|-----------|------|
| Wire (기본) | 수직 1발 | 천장까지 고정 유지 |
| Double Wire | 수직 2발 동시 | 좌우 2방향 동시 발사 |
| Power Wire | 수직 1발 | 즉시 수거 가능, 재발사 속도 빠름 |
| Vulcan Missile | 수직 연사 | 고정 없이 관통 후 소멸 |

---

## 파워업 아이템

풍선 처치 시 파워업 아이템이 드롭될 수 있다.

| 아이템 | 효과 | 지속 시간 |
|--------|------|-----------|
| 모래시계 | 풍선 이동 속도 감소 | 일정 시간 |
| 시계 | 풍선 완전 동결 | 일정 시간 |
| 다이너마이트 | 화면 모든 풍선을 극소 크기로 축소 | 즉시 |
| 포스 필드 | 플레이어 무적 | 일정 시간 |

---

## 점수 시스템

| 행동 | 점수 |
|------|------|
| 극소 풍선 처치 | 가장 높음 |
| 소 풍선 처치 | 높음 |
| 중 풍선 처치 | 보통 |
| 대 풍선 처치 | 낮음 |
| 아이템 획득 | 보너스 |

> 크기가 작을수록 처치 점수가 높다.

---

## 라이프 및 게임 오버

- 시작 시 라이프 **3개**
- 어떤 크기의 풍선에 닿아도 라이프 **1 감소**
- 라이프 소진 시 **게임 오버**
- 게임 오버 후 메인 화면으로 복귀

---

## 스테이지 클리어

- 화면의 **모든 풍선 소멸** 시 클리어
- 클리어 후 다음 스테이지로 자동 이동
- 마지막 스테이지 클리어 시 **엔딩 화면** (추후 구현)
