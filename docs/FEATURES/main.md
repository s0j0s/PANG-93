# 메인 화면 구성

## 화면 레이아웃

```
┌─────────────────────────────────┐
│  🎈     🎈          🎈         │  ← 풍선 애니메이션 배경
│                                 │
│           P  A  N  G            │  ← 게임 타이틀
│                                 │
│      ▶ [ GAME START ]           │  ← 선택된 항목 (▶ 표시)
│        [ HOW TO PLAY ]          │
│        [ QUIT        ]          │
│                                 │
│  🎈          🎈      🎈        │
└─────────────────────────────────┘
```

## 구성 요소

### 타이틀
- 화면 상단 중앙에 게임 이름 **PANG** 표시
- 픽셀 레트로 폰트 (`Press Start 2P`) 적용
- 골드 컬러(`#FFD700`) + 글로우 효과

### 메뉴 항목
| 항목 | 동작 |
|------|------|
| GAME START | Mission 1 Stage 1로 이동 (Phase 2에서 구현) |
| HOW TO PLAY | 조작 방법 안내 모달 표시 |
| QUIT | 종료 확인 후 `window.close()` |

### 조작 방법 (메뉴 내비게이션)

| 입력 | 동작 |
|------|------|
| `↑` / `↓` 방향키 | 메뉴 항목 위아래 이동 |
| `Enter` | 현재 선택된 항목 실행 |
| 마우스 클릭 | 해당 항목 바로 실행 |
| 마우스 오버 | 해당 항목으로 선택 이동 |

- 선택된 메뉴 항목에는 좌측에 `▶` 커서 표시
- 선택 항목은 골드 컬러(`#FFD700`)로 하이라이트
- 첫 진입 시 `GAME START`가 기본 선택

## 배경 풍선 애니메이션
- 8개의 풍선이 위아래로 천천히 떠다니는 CSS 애니메이션
- 풍선마다 크기·색상·속도·딜레이가 다름
- 장식용이며 클릭·충돌 판정 없음

## 진입 흐름

```
메인 화면 진입 → GAME START 선택(↑↓ 또는 마우스) → Enter 또는 클릭 → 게임 시작
```

## HOW TO PLAY 모달

```
┌──────────────────────────────────┐
│         HOW TO PLAY              │
│                                  │
│  이동    ← → 방향키              │
│  발사    SPACE                   │
│                                  │
│  풍선을 모두 터뜨리면 클리어      │
│  풍선에 닿으면 라이프 1 감소      │
│  라이프 0 → 게임 오버             │
│                                  │
│           [ 닫기 ]               │
└──────────────────────────────────┘
```

- `ESC` 키로도 닫기 가능
- 배경 딤 처리 (반투명 오버레이)
