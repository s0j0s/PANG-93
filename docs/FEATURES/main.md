# 메인 화면 구성

## 화면 레이아웃

```
┌─────────────────────────────────┐
│         PANG                    │  ← 게임 타이틀
│                                 │
│       [ GAME START ]            │  ← 1인 플레이 시작
│       [ 2P START   ]            │  ← 2인 협력 시작 (추후 구현)
│                                 │
│       [ HIGH SCORE ]            │  ← 최고 점수 확인 (추후 구현)
│                                 │
│  ♦ 조작 방법 ♦                   │
│  ← → : 이동   SPACE : 작살 발사   │
└─────────────────────────────────┘
```

## 구성 요소

### 타이틀
- 화면 상단 중앙에 게임 이름 **PANG** 표시
- 애니메이션 풍선이 배경에서 튀어다니는 연출 (선택 사항)

### 메뉴 항목
| 항목 | 동작 |
|------|------|
| GAME START | Mission 1 Stage 1 로 이동 |
| 2P START | (추후 구현) |
| HIGH SCORE | (추후 구현) |

### 조작 안내
- 키보드 조작 방법을 하단에 간략히 표시
- `←` `→` 이동 / `Space` 발사

## 진입 흐름

```
메인 화면 → [GAME START 클릭] → Mission 1 Stage 1 시작
```

## 상태

- 메인 화면에서는 BGM 재생 (선택 사항)
- 메뉴 항목 선택 시 하이라이트 표시
